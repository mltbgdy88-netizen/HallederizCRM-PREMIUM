# wallpaper-sector

Duvar kağıdı ve dekorasyon sektörüne özel modüller burada geliştirilecektir.

İlk kapsam:

- Rulo hesabı
- Metrekare / rulo dönüşümü
- Fire hesabı
- Oda / duvar ölçüsü
- Alan hesaplayıcı
- Numune defteri
- Katalog defteri
- Askılı stant
- Bayiye verilen numune/katalog takibi
- Usta / uygulama ekibi
- Uygulama garanti kaydı

Bu alanlar açık kaynak CRM repolarında doğrudan hazır değildir; özel domain modülü olarak geliştirilecektir.
