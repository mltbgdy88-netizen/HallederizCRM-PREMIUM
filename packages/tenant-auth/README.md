# tenant-auth

Workspace, kullanıcı, rol ve yetki altyapısı burada geliştirilecektir.

İlk kapsam:

- Workspace / tenant
- User
- Session
- Role
- Permission
- Role-permission mapping
- Record ownership
- Audit log hooks

Kabul standardı:

- Her tenant verisi ayrılmalı.
- Kullanıcı sadece yetkili olduğu kayıtları görmeli.
- Yönetici rolleri merkezi olarak yönetilebilmeli.
