# inventory

Ürün, stok, depo, şube ve bayi stok görünürlüğü burada geliştirilecektir.

İlk kapsam:

- Product
- SKU
- Product category
- Product variant
- Warehouse
- Branch stock
- Dealer stock visibility
- Reserved stock
- Critical stock alerts

Kaynak referansları:

- OpenSource Startup CRM ürün/katalog referansı
- NocoBase tablo/form/workflow yaklaşımı
