# crm-core

Bu paket birleşik CRM ürününün ana domain paketidir.

İlk hedef entity'ler:

- Workspace
- User
- Role
- Permission
- Customer
- Company
- Contact
- Lead
- Deal
- Task
- Note
- Attachment

Kaynak referansları:

- Atomic CRM: UI ve demo akışları
- Django CRM: CRM domain yaklaşımı
- Krayin CRM: satış süreçleri
- NocoBase: low-code modelleme yaklaşımı
