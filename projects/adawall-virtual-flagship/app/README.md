# ADAWALL Virtual Flagship App

Bu uygulama, ADAWALL Virtual Flagship — T Plan Wallpaper Corridor pilotunun ilk teknik prototipidir.

## Hedef

- T plan fallback mağaza sahnesi
- Wallpaper Gallery Corridor
- 1 pilot loca
- 3 renk varyasyonu
- Sağ ürün paneli
- Mock bayi stoğu ve fabrika stoğu

## Kurulum

```bash
npm install
npm run dev
```

## Kontrol Komutları

```bash
npm run typecheck
npm run build
npm run lint
```

## Not

Bu prototipte gerçek GLB mekan dosyası henüz yoktur. Uygulama fallback geometri ile açılacak şekilde tasarlanmıştır. Final fotogerçekçi mekan modeli `public/models/flagship-shell.glb` dosyasıyla eklenecektir.
