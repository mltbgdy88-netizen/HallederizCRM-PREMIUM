# ADAWALL Virtual Flagship

Bu klasör, Adawall için planlanan fotogerçekçi sanal mağaza / 3D showroom SaaS pilotunun proje alanıdır.

## Kilitlenen Pilot Karar

**Proje adı:** ADAWALL Virtual Flagship — T Plan Wallpaper Corridor

Ana mağaza mimarisi korunur. Ana mağazaya T şeklinde bağlanan geniş ve uzun bir Wallpaper Gallery Corridor eklenir. İlk üretim hedefi ürünlerle doldurulmuş final sahne değil; gerçek çekim sanal tur hissi veren ürünsüz / az ürünlü fotogerçekçi mekan kabuğudur.

## Ana Hedef

Adawall bayilerinin kullanabileceği, tema/mekan bazlı, ürün, fiyat ve stok verileriyle çalışabilecek SaaS altyapısına dönüşebilecek sanal mağaza platformu.

## İlk Faz

1. Mekan kilit dokümanı
2. Fotogerçekçi boş mekan brief'i
3. 1 loca prototipi
4. Desen + renk varyasyonu sistemi
5. Ürün kartı, bayi stoğu ve fabrika stoğu veri modeli

## Araçlar

- Cursor: ilk kod üretimi, component yapısı, UI, R3F sahne prototipi
- Codex: kod denetimi, refactor, test, kalite kontrol
- Blender / 3D modelleme: T plan mekan, corridor, loca sistemi, GLB export
