# Roadmap

## Phase 1 - UI lock

Status: active.

Goals:
- Accept main UI language.
- Verify key screens by screenshots.
- Fix only visible UI issues.
- Keep backend work out of scope.

Key screens:
- Dashboard
- Hızlı İşlem
- WhatsApp
- Ayarlar
- Onaylar
- Stok
- Depo Hazırlık
- Depo Hazırlık Fişi

## Phase 2 - Modal and window polish

Start after main UI lock.

Goals:
- Review detail modals.
- Review drawer and popup language.
- Align spacing, buttons, headers, and empty states.

## Phase 3 - Engine and integrations

Start after UI lock.

Goals:
- WhatsApp QR connection.
- Webhook settings.
- Supplier and factory API connections.
- Product and stock sync.
- Order automation paths.
- AI approval engine.

## Phase 4 - Archive and documents

Goals:
- PDF archive records.
- Warehouse prep document archive.
- Product delivery document archive.
- Searchable operation history.
