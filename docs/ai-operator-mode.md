# AI Operator Mode

## Modlar
- Read-only: analiz/ozet/yorum, mutation yok.
- Operator: mutation niyeti proposal uretir, dogrudan execution yapmaz.

## Zincir
1. Kullanici komutu
2. AI chat cevabi
3. Mutation niyeti varsa proposal
4. Approval
5. Approval execution dispatcher
6. Audit/timeline kaydi

## Guvenlik
- Approval olmadan mutation execution yok.
- Yetkisiz kullanicilar confirm/reject aksiyonlarina erisemez.
